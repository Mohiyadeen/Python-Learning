Import in Python:

What Can You Import?
The terminology can get confusing, especially when learning from multiple sources, so here’s a quick list of what can be imported and what they’re called.

1. Module: simply a file with a .py extension.
2. Package: a directory containing an __init__.py file and normally other modules.
3. Built-in Module: A module that is natively installed with Python.
4. Object: Anything inside a module/package that can be referenced such as a class, function, or variable.
